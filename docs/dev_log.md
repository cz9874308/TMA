## 新手教程

### 1 - 监控（涨停/跌停）板（买一/卖一）挂单金额

```python
import tma
from tma.monitor import sm_limit

# 设置SCKEY - 调用Server酱推送消息到微信
tma.SCKEY = "XXXXXXXXXX"

sm_limit('600122', kind="zt", threshold=10000, interval=1)

```
